# AI 味诊断与替换规则

## 总原则

AI 味通常不是“语法错误”，而是文本太像模板、太像翻译、太像报告、太缺少人说话时的现场感。

去 AI 味时，不要只做同义词替换。要同时处理：

- 词：删掉套话、虚词、宏大词。
- 句：拆长句，减少被动句和名词化。
- 段：让推进方式更像人讲述，而不是像报告目录。
- 例子：把英文语料习惯里的例子换成中文语境。
- 风格：保留用户的表达习惯，不要统一改成“标准口语文案”。

## 一、常见 AI 味类型

### 1. 开场宏大叙事

高风险表达：

- 在当今快速发展的时代
- 随着科技的不断进步
- 在数字化浪潮的推动下
- 近年来，……逐渐成为焦点
- 如今，……已经渗透到我们生活的方方面面

问题：太像作文开头或行业报告，缺少具体场景。

替换方向：直接进入问题、场景或冲突。

示例：

```text
AI味：在当今快速发展的时代，人工智能正在深刻改变内容创作行业。
真人感：你会发现，这两年写稿这件事变了。以前憋半天写不出开头，现在 AI 先把大纲给你铺出来了。
```

### 2. 书面化身份表达

高风险表达：

- 作为一名深耕某领域多年的从业者
- 我长期关注并研究某领域
- 凭借多年行业经验
- 站在行业发展的角度来看

替换方向：换成更直接的人话。

```text
AI味：作为一名深耕 AI 领域多年的老兵，我对此有着深刻理解。
真人感：说白了，我干 AI 这事已经挺多年了，有些坑我是真踩过。
```

### 3. 翻译腔

高风险表达：

- 这使得我们能够……
- 它为我们提供了一个机会去……
- 以一种更加高效的方式……
- 被广泛认为是……
- 这背后反映的是……
- 从某种意义上来说……
- 这标志着……的开始

替换方向：主动句、短句、直接说明。

```text
AI味：这使得创作者能够以一种更加高效的方式完成内容生产。
真人感：创作者不用再从零开始憋稿了，先让 AI 打个底，自己再改。
```

### 4. 抽象名词堆叠

高风险表达：

- 系统性提升
- 底层逻辑
- 能力边界
- 价值闭环
- 生态赋能
- 路径依赖
- 增长飞轮
- 内容生产效率
- 多维度协同
- 全链路优化

这些词不是绝对不能用，但连续出现会让文本像咨询报告。

替换方向：把名词变成动作，把概念变成画面。

```text
AI味：这会带来内容生产效率的系统性提升。
真人感：最直接的变化就是，写一篇稿子不再从空白页开始。
```

### 5. 万能总结句

高风险表达：

- 这不仅是……更是……
- 它不仅改变了……也重塑了……
- 这既是机遇，也是挑战。
- 未来已来。
- 我们需要以更加开放的心态拥抱变化。
- 只有这样，才能在时代浪潮中立于不败之地。

替换方向：具体收束，回到用户能感知的结果。

```text
AI味：这不仅提升了效率，更重塑了内容行业的未来格局。
真人感：所以你会发现，真正被改变的不是“行业格局”这四个字，而是你今天晚上能不能少熬两个小时。
```

### 6. 机械排比

高风险结构：

```text
它可以帮助我们提高效率；
它可以帮助我们降低成本；
它可以帮助我们优化体验；
它可以帮助我们实现增长。
```

替换方向：把列表变成有轻重的讲述。

```text
真人感：它最有用的地方其实就两点：一个是帮你省时间，另一个是帮你先把最难开始的那一步跨过去。至于“增长”“体验”这些词，后面再说。
```

### 7. 假口语和强行网感

高风险表达：

- 家人们谁懂啊
- 真的绝绝子
- 直接封神
- 太炸裂了
- 赢麻了
- 普通人一定要抓住
- 看懂的人已经开始行动了
- 这波不冲真的亏

问题：看似口语，其实很像模板化营销号。

替换方向：真实一点，少夸张，多具体。

```text
AI味：这个工具真的太炸裂了，普通人一定要抓住机会。
真人感：这个工具有没有那么神？我觉得没必要吹太满。但如果你经常写稿，它确实能帮你省不少时间。
```

### 8. 英文语料式比喻

高风险表达：

- 像吃了二十个汉堡一样重
- 就像一台精密运转的机器
- 就像硅谷车库里的创业故事
- 就像星巴克重新定义咖啡体验
- 就像沃尔玛改变零售行业

替换方向：换成中文生活经验或中文商业环境。

```text
AI味：这就像星巴克改变了人们喝咖啡的方式。
真人感：这有点像当年大家从路边小店买奶茶，突然变成排队买喜茶、瑞幸券、抖音团购那套逻辑。
```

| 打造 | 营销味 | 做出、搭出、建立 |

| 触达 | 商业黑话 | 让用户看到 / 联系到 |

## 三、四种改写手法

### 1. 抽象词落地

把“行业变革”落到具体场景。

```text
原句：AI 正在推动知识工作的效率革命。
改法：以前你写一份方案，可能先盯着空白文档半小时。现在 AI 至少能先给你铺个框架，你再往里面改。
```

### 2. 名词改动词

```text
原句：这会实现用户体验的优化。
改法：用户用起来会少卡几步，少点几次按钮。
```

### 3. 被动改主动

```text
原句：这一趋势被越来越多的创作者所接受。
改法：现在很多创作者已经开始这么干了。
```

### 4. 总结改现场

```text
原句：这体现了内容创作方式的深层变化。
改法：最明显的变化就是，写稿不再是一个人硬憋，而是先有人帮你把草稿搭起来。
```

## 四、保留一点“不完美”

真人表达不需要每句话都绝对精修。以下元素可以适量保留：

- “其实”“说白了”“你看”“问题是”
- 轻微重复，比如“这个事，这个事真正难的地方在……”
- 自然追问，比如“为什么？很简单。”
- 口语转折，比如“但问题来了”
- 稍微不对称的句式

但不要过量，否则会显得啰嗦。

## 五、诊断输出模板

如果用户要求先诊断，可以使用：

```markdown
## AI 味诊断

2. **翻译腔**：……

4. **不符合中文语境的例子**：……
5. **读起来拗口的句子**：……

## 改写方向

- ……
- ……
```
